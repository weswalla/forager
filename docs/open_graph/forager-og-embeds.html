Thumbnail · 04 · dynamic

Shared trail — a social object

Generated per trail. The account doing the sharing leads: avatar, display name, and ATProto handle up top, then the trail title as hero, then the trail spine itself (nodes coloured by connection type). Sample values shown.

Shared trail/t/<trail-id> · per-trail shareforager-og-trail-shared.png

Fields the server fills per trail

  • Avatar — the account’s ATProto avatar (<image>); falls back to a tinted monogram when none is set.
  • Display name — e.g. Ronen Tamari.
  • Handle — the Bluesky/ATProto handle, e.g. @ronen.bsky.social.
  • Trail title — the hero line; single line reads best (wrap or truncate past ~26 characters at this size).
  • Subtitle / stop count — a one-line summary ending in “N stops”.
  • Spine nodes — optional: colour a few by connection type (mutual · similar · connected), the rest neutral.

A ready-to-fill forager-og-trail-shared.template.svg is attached, with {{DISPLAY_NAME}}, {{HANDLE}}, {{TRAIL_TITLE}}, {{TRAIL_SUBTITLE}}, {{INITIAL}} and {{AVATAR_TINT}} placeholders — render it to PNG on save, or on the fly behind /og/t/<id>.png.

Drop-in

Meta block

Pairs title 1 + description 1 + the roundel image. Swap the og:image filename per page to use a different card.

<meta property="og:type" content="website">
<meta property="og:site_name" content="Forager">
<meta property="og:url" content="https://forager.link/">
<meta property="og:title" content="Forager — a calm client for Semble">
<meta property="og:description" content="A calm client for the Semble network. Forage the links worth keeping, add notes, and follow the trails between them — no feed, just paths.">
<meta property="og:image" content="https://forager.link/og/forager-og-roundel.png">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:image:alt" content="The Forager bear peeking over a trail">
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Forager — a calm client for Semble">
<meta name="twitter:description" content="A calm client for the Semble network. Forage the links worth keeping, add notes, and follow the trails between them — no feed, just paths.">
<meta name="twitter:image" content="https://forager.link/og/forager-og-roundel.png">

Two gotchas. Use an absolute https URL for og:image (scrapers can’t resolve relative paths), and add a cache-buster like ?v=2 when you change an image — Slack, Facebook and others cache hard.
