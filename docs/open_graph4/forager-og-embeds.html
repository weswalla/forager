Per-trail meta

Dynamic meta for a shared trail

<meta property="og:type" content="article">
<meta property="og:site_name" content="Forager">
<meta property="og:url" content="https://forager.link/t/{{trail_id}}">
<meta property="og:title" content="{{trail_title}} — a trail on Forager">
<meta property="og:description" content="Curated by {{display_name}} ({{handle}}) · {{n}} stops on Semble">
<meta property="og:image" content="https://forager.link/og/t/{{trail_id}}.png">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta name="twitter:card" content="summary_large_image">
<meta property="article:author" content="{{handle}}">

Two gotchas. Use an absolute https URL for og:image, and add a cache-buster like ?v=2 when you change an image — Slack, Facebook and others cache hard.
